Доступные программы:

Tetris [tetris.bat] - игра Тетрис на Batch, немного изменена под OldOS (добавлена опция выхода)
MyNotes [mynotes.bat] - простая программа для быстрого создания мини-заметок в одну строчку. Написана создателями OldOS.
Календарь [calendar.bat] - календарь на Batch! Написана создателями OldOS. Имеет возможность отображать года в диапазоне с 1901 по 2112. Начиная с сборки от 30 ноября 2025 он предустановлен.
5 ночей со странами [5nc.bat] - пять ночей со странами!
OldChat [oldchat.bat] - первое коммуникационное приложение для OldOS, увы, поддерживает только один общий чат, пока нет регистрации.
OldNETER [oldnet.bat] - браузер для просмотра OldNET - сети сайтов, предназначенных для OldOS.
OldMySite - программа для создания сайтов OldNET - В РАЗРАБОТКЕ
TimeMachine [timemachine.bat] - создание бэкапов для OldOS.

Позже для установки, введите название в квадратных скобок, с припиской .bat
Вводите без самих квадратных скобок.
